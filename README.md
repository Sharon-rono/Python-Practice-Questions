## Python Questions
A collection of Python exercises covering fundamental data structures and control flow, completed in Google Colab.  
## What's Included
 Data Structures:
 
 Dictionaries - creating, updating, removing keys, iteration  
 Lists - manipulation, user input, duplicate detection  
 Tuples - understanding immutability  
 Sets - operations (union, intersection, difference), removing duplicates  

 ## How to Use
 
 -Dowload the file  
 -Run cells sequentially using Shift + Enter  
 -Input values when prompted for interactive exercises  
 -Experiment by modifying the code
